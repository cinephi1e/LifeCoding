생활코딩

HTML: 1~12강

* html의 개념: 웹페이지를 구성하는 뼈대

* html이 중요한 이유

  1. 검색결과에서 경쟁력 획득
  2. 시각장애인을 위한 정보제공
* 폰트 스타일
  • u: 밑줄
  • strong: 진하게
* 이미지
img src="이미지 주소" + width="px값 혹은 % 입력"

* 줄 바꿈
  • br: 줄바꿈만 부여. 닫힘 태그X
  • p: 단락 지정. 닫힘 태그O
* 부모자식과 목록
  1. 부모(상위)
    2. - ol(Oldered List): 순번 자동부여
    - ul(Unoldered List) 순번 없이 도형 부여
  2. 자식(하): li 리스트 나열
* CSS
attribute 부분에 style="효과+값"
ex) p style="font-size:20px"
